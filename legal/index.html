CloudFlare
This website integrates components of CloudFlare.
CloudFlare is used as:
  • a CDN (Content-Delivery-Network); One place for getting the entire website
  • a cache for faster response-time
  • an optimization platform (Rocket Loader; See below) for optimized builds
  • a domain and DNS controller, used for faster routing
  • as a server shield against attacks
To provide these features, all traffic is routed through CloudFlare. CloudFlare itself uses several technologies to shield and optimize, such as their Rocket Loader.
Additionally, CloudFlare is collecting minimal, anonymised, usage data of all traffic passing through them. This data is only kept for a limited time frame and only visible to the owner. Although, the same data can be collected on the servers.
For further information, please consider CloudFlare's privacy policy.
