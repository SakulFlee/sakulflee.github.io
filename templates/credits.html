Building blocks of this website

Open-Source

The entire source code of my website is Open-Source, licensed under MIT and can be found on my repository.

Rocket.rs

Backend

Rocket.rs is used for the backend. All required files for the frontend are served by Rocket.rs and required data is fetched from the database.

Rocket.rs is not only a server and client library / crate, but comes powered with a lot of useful features, tools and libraries / crates included!

Bulma

CSS Framework

This website is designed with Bulma, a free, open source CSS framework based on Flexbox.

Bulma makes it extremely easy to design responsive (flex) user-interfaces. It is even possible to create websites without own stylesheets.

I still added some things myself and modified some of the component, but this websites design-core is Bulma. If you are interested in my changes check out the style-sheets on my repository.

Diesel.rs

Data-Layer

Diesel.rs is used for the data-layer. Diesel.rs is not a database on its own, but can be used to connect to different databases and make queries easier.

Diesel.rs makes it incredibly easy to create a schema and use it without restrictions. Diesel.rs also integrates with Rocket.rs

Tera

Template Engine

Tera is used as a template engine for the frontend.

Using templates makes it possible to reuse a lot of code and split the website into multiple parts / components, which then only need to be combined.

Tera also includes quite powerful tools and a sort-of scripting language

Font Awesome

Icons

This website makes use of different icons.
Most, if not all, are from Font Awesome.

Font Awesome, in its free license model, requires you to give proper credit to all used icons and state their changes.

I did introduce slight optimization on some icons, but the overall look of the icon did not change.
Furthermore, some icon colors where changed to better fit the color-schema of my website.

The license can be found here.

Rust crates

This entire website is build with Rust.

Even the client-library is written in Rust and compiled to WASM.

There are soooooo many different crates I am using, and even more crates that other crates depend on. Currently, there are, per-project, over 200 crates in use. If you want to know more check out the repository and compile the code yourself!
